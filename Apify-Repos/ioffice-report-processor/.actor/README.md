## iOffice Reports Processor

This actor will download the report data from iOffice site and build the dataset based on given inputs.


## Steps

- Login to iOffice site : https://ioffice.rogerspos.com/office/login.php
- On dashboard https://ioffice.rogerspos.com/office/, click on `Tools` menu
- On left side nav, click on `Custom Reports` menu
- On right side pane, click on `Saved Reports` 
- Find particular report name in the list. e.g. `TEK2020ALLDATA`
- Click on Next
- Select Start Date and End Date along with Selecting all stores
- Click on `Generate report`
